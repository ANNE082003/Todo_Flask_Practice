# Todo_Flask_Practice
## Languages/Components Used

- Frontend: Basic HTML,Bootstrap
- Backend: Python
- Database: MySQL

## Installation



Install dependencies

```
pip install -r requirements.txt
```

Run Flask Server

```
python run.py
```

## Functions

- create,view,edit and delete Todos
- Create Todo List
## Technical Domain

- Create using Python
- Flask for the Web
- SQLAlchemy for Database
- Bootstrap for the UI
